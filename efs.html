EFS - Don du sang

L'objectif du projet a été de développer dans le cadre d'un WorkShop en fin de première année de BTS SIO à l'école Sciences U. L'objectif était de réaliser pour l'Établissement Français du Sang une application permettant aux utilisateurs de se connecter, vérifier s'ils sont éligibles aux dons du sang, et prendre rendez-vous.

Ce projet a été réalisé dans le cadre d'un WorkShop de mon BTS Services Informatiques aux Organisations à l'école Sciences-U Lyon. L'objectif étant d'apprendre à travailler en groupe, se répartir les tâches, incorporer des données et informations d'un site externe.

Compétence utilisé :

  • Travail d'équipe

    Les tâches et le développement de cette application a nécessité l'entraide ainsi que le partage de code afin que chacun dans le groupe s'occupe de certain partie

  • Technologies

    PHP, SQL, HTML, CSS, JS -> Nous aurons utilisé ces 5 languages dans le développement de ce projet

  • Outils

    Comme outils nous aurons utilisé GitHub pour le partage de code et de pouvoir suivre l'avancement de tout le monde Discord pour la communication Visual Studio Code pour la partie développement

Galaxy Swiss Bourdin

L'objectif du projet a été de développer une application pour le groupe GSB permettant aux utilisateurs de prévoir des visites avec des médecins dans le but de leur faire essayer des échantillons de médicaments, puis de rédiger des comptes-rendus de ces visites.

Ce projet a été réalisé dans le cadre de l'épreuve E5 de mon BTS Services Informatiques aux Organisations à l'école Sciences-U Lyon. L'objectif étant de mettre en application les connaissances acquérises au cours des 2 années pour réaliser un projet concret, complet et fonctionnel.

Compétence utilisé :

  • Travail d'équipe

    Les tâches et le développement de cette application a nécessité l'entraide ainsi que le partage de code afin que chacun dans le groupe s'occupe de certain partie

  • Technologies

    PHP, SQL, HTML, CSS, JS -> Nous aurons utilisé ces 5 languages dans le développement de ce projet

  • Outils

    Comme outils nous aurons utilisé GitHub pour le partage de code et de pouvoir suivre l'avancement de tout le monde Discord pour la communication Visual Studio Code pour la partie développement
